<h1>Droplet fusion analysis</h1><br>
<b>Author</b>: Archishman Ghosh<br>
<b>Key words</b>: droplet fusion, Stretched exponential, Fitting<br>
<b>Research subjects</b>: Phase separation<br>
<br>
<p>Protein droplet fusion can be tracked using the force as a signal. When a droplet from trap 1 fuses onto trap 2, the force profile on trap 2 is normalized and fit to a stretched exponential function. The relaxation time Tau is dependent on viscosity and surface tension.</p><p><br></p><p>Transfer exported force data (h5 file) into folder containing script and run in MATLAB.</p><p><br></p><p>Details in:&nbsp;<a href="https://onlinelibrary.wiley.com/doi/10.1002/anie.202006711">https://onlinelibrary.wiley.com/doi/10.1002/anie.202006711</a>â€‹</p><br><br>
